# maxxfetch
simple and stupid fetch tool developed entirely by me ^~^

maxxfetch is distribuited on the GNU/GPLv3 license

compiling instructions: run "sudo/doas make install" to compile maxxfetch.cpp. maxxfetch will get copied to /usr/local/bin after compiling so you can run it in your shell
